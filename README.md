# 깃깃 (GitGit)

현대적인 웹 기술을 활용한 프로젝트 저장소입니다.

## 🚀 개요

이 프로젝트는 프리미엄 디자인과 사용자 경험을 최우선으로 생각하는 웹 애플리케이션의 기초를 제공합니다.

## ✨ 주요 특징

- **Modern UI**: 글래스모피즘(Glassmorphism)과 다크 모드 기반의 세련된 디자인.
- **Responsive**: 데스크탑, 태블릿, 모바일 등 모든 환경에 최적화된 레이아웃.
- **Premium Aesthetics**: 유려한 그라데이션과 인터랙티브한 요소들.
- **Sub Pages**: 프로젝트 상세 정보를 담은 `sub1.html` 포함.

## 🛠 사용된 기술 stack

- **HTML5**: 시맨틱 마크업을 통한 SEO 최적화.
- **CSS3**: 변수(Variables), Flexbox/Grid, 애니메이션을 활용한 스타일링.
- **Google Fonts**: Inter 및 Outfit 폰트를 활용한 타이포그래피.

## 📦 시작하기

`index.html` 파일을 브라우저에서 열어 페이지를 확인할 수 있습니다.
